Upload Inventory CSV

Upload a CSV file to create or update items. The merge strategy uses (Item Name + Container Name) as the unique key.

CSV Format:

Required Columns:

Item Name, Category, Quantity, Container Name, Location, Dropbox URL

Extended Format (Optional columns):

Item Name, Category, Quantity, Container Name, Location, Brand/Platform, Serial Number, Estimated Value, Dropbox URL, Image URL, Low Stock Threshold

Container Name = the physical box/case the QR code is on (e.g., "Milwaukee Packout A").
Location = where the container lives (e.g., "Truck 1", "Shop", "Warehouse").
